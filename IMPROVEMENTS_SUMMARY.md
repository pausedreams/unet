# MK-UNet 模型优化改进总结

## 🎯 优化目标
根据 MK-UNet 论文内容，系统性优化模型训练策略和数据处理流程，提升 Dice 分数。

---

## ✅ 已完成的优化改进

### 1. **训练超参数优化** (基于论文推荐)

#### 学习率调整
- **改进前**: `5e-4`
- **改进后**: `1e-3` (根据论文推荐)
- **原因**: 论文使用更高的初始学习率，配合余弦退火调度器可以快速收敛到最优解

#### 混合精度训练 (AMP)
- **新增功能**: 启用自动混合精度训练
- **配置**: `USE_AMP = True`
- **优势**:
  - 节省约 50% 显存
  - 加速训练速度（约 20-30%）
  - 支持更大的有效 batch size

### 2. **优化器与学习率调度器**

#### AdamW 优化器
- **配置**: `weight_decay=1e-4`
- **优势**: 相比 Adam，AdamW 通过解耦权重衰减提供更好的正则化效果

#### 余弦退火学习率调度器
- **配置**: `CosineAnnealingLR(T_max=60, eta_min=1e-6)`
- **优势**: 
  - 平滑的学习率衰减曲线
  - 避免学习率突降导致的性能下降
  - 有助于模型在训练后期找到更优的最小值点

### 3. **梯度累积策略**

- **配置**: `GRADIENT_ACCUMULATION_STEPS = 2`
- **效果**: 
  - 实际 batch size = 4 × 2 = 8
  - 更稳定的梯度更新
  - 在不增加显存占用的情况下模拟更大 batch size

### 4. **数据增强增强** (基于论文多尺度训练思想)

#### 随机旋转
- **角度范围**: `-20° ~ +20°` (扩大角度范围)
- **目的**: 增加样本多样性，提升模型泛化能力

#### 随机缩放 (多尺度训练)
- **缩放范围**: `90% ~ 110%`
- **实现方式**: 
  1. 随机放大图像到 256×(0.9~1.1)
  2. 随机裁剪回 256×256
- **目的**: 模拟论文中的多尺度训练策略，让模型适应不同尺寸的目标

#### 颜色抖动增强
- **新增饱和度调整**: `random.uniform(0.9, 1.1)`
- **亮度调整**: `random.uniform(0.9, 1.1)`
- **对比度调整**: `random.uniform(0.9, 1.1)`
- **目的**: 增强模型对不同成像条件的鲁棒性

### 5. **损失函数优化**

#### Focal Loss + Dice Loss 联合损失
- **Focal Loss 参数**: `alpha=0.75, gamma=2.0`
- **权重分配**: Focal Loss 50% + Dice Loss 50%
- **优势**:
  - Focal Loss 专注于难分类的像素
  - 解决前景 - 背景类别不平衡问题
  - Dice Loss 直接优化分割指标

#### 深度监督联合损失
- **权重分配**: 
  - 主输出: 60%
  - 辅助输出 1: 20%
  - 辅助输出 2: 20%
- **优势**: 
  - 多个监督信号引导特征学习
  - 改善梯度流动
  - 提升深层网络的训练效果

### 6. **训练轮数与早停策略**

- **训练轮数**: `60 epochs` (增加 50%)
- **早停耐心值**: `10 epochs` (增加 25%)
- **目的**: 
  - 充分训练模型直到收敛
  - 避免过早停止导致欠拟合

## 🔬 技术细节

### 为什么选择这些改进？

1. **混合精度训练**: 论文提到计算资源受限，AMP 可以显著降低显存占用
2. **梯度累积**: 模拟更大 batch size，提升训练稳定性
3. **Focal Loss**: 医学图像中前景（肿瘤）通常远小于背景，类别不平衡严重
4. **多尺度数据增强**: 论文核心思想之一，提升模型对不同尺寸目标的适应性
5. **深度监督**: 论文关键创新点，通过多个辅助头提供额外监督信号

### 改进的科学依据

所有改进都基于：
- MK-UNet 论文的官方推荐配置
- 医学图像分割领域的最佳实践
- 深度学习训练的通用原则

## ⚠️ 注意事项

1. **GPU 显存要求**: 
   - 最低：8GB (启用 AMP 后)
   - 推荐：12GB 或更高

2. **训练时间**:
   - 预计完成 60 轮需要 8-12 小时（取决于 GPU 性能）

3. **过拟合风险**:
   - 如果验证集 Dice 不再提升，可以提前停止训练
   - 或者增加数据增强强度

4. **超参数调优**:
   - 如果效果不理想，可以尝试调整学习率、Focal Loss 的 alpha/gamma 参数

## 📝 参考论文

Rahman, M. M., & Marculescu, R. (2025). MK-UNet: Multi-kernel Lightweight CNN for Medical Image Segmentation. arXiv preprint arXiv:2509.18493.

**论文关键发现**:
- 多核深度卷积 (MKDC) 比单核卷积更有效
- 轻量级注意力机制 (CA + SA) 只在 decoder 中使用
- 深度监督显著提升分割精度
- AdamW + 余弦退火是最佳优化组合
